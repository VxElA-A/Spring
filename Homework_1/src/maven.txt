1.Создание нового Maven проекта:

mvn archetype:generate
Используется для создания нового Maven проекта на основе архетипа.
----------------------------------------------------------------
2.Сборка проекта:

mvn package
Эта команда компилирует проект и упаковывает его в JAR-файл (или WAR-файл, если это веб-приложение).
----------------------------------------------------------------
3.Компиляция проекта:

mvn compile
Компилирует исходный код проекта.
----------------------------------------------------------------
4.Запуск тестов:

mvn test
Запускает тесты проекта.
----------------------------------------------------------------
4.Очистка проекта:

mvn clean
Удаляет скомпилированные файлы и директории (target), очищая проект.
----------------------------------------------------------------
5.Инсталляция в локальный репозиторий:

mvn install
Сборка проекта и установка его в локальный репозиторий Maven.
----------------------------------------------------------------
6.Сборка проекта с пропуском тестов:

mvn package -DskipTests
Компилирует и упаковывает проект, пропуская выполнение тестов.
----------------------------------------------------------------
7.Запуск приложения Spring Boot:

mvn spring-boot:run
Если вы используете Spring Boot, эта команда запускает приложение.
----------------------------------------------------------------
8.Обновление зависимостей:

mvn dependency:resolve
Разрешает и загружает все зависимости проекта.
----------------------------------------------------------------
9.Создание сайта проекта:

mvn site
Создает отчет о проекте, включая документацию и метрики.
----------------------------------------------------------------
10.Запуск Jetty серверa:

mvn jetty:run
Если у вас есть веб-приложение, эта команда запускает его на встроенном Jetty сервере.
----------------------------------------------------------------
11.Просмотр дерева зависимостей:

mvn dependency:tree
Показывает дерево зависимостей проекта.
----------------------------------------------------------------
12.Генерация исходников из XSD схемы:

mvn jaxb2:xjc
Эта команда генерирует Java-классы из XSD схемы (требует соответствующий плагин).
